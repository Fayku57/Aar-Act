## 📋 About Aar-Act

Aar-Act is a volunteer-led project by CyberAar focused on improving security for Senegal's critical systems. It gathers and shares best practices to help protect important infrastructure from cyber threats. The goal is to support local teams and the diaspora in building stronger defenses against risks like ransomware and data breaches.

This tool is open source and anyone can contribute. It targets those interested in cybersecurity for African infrastructure, especially Senegal. Aar-Act serves as a practical guide and toolkit that assists administrators and concerned citizens in applying proven security steps.

## 💻 System Requirements

Before you download Aar-Act, make sure your device meets these requirements:

- Windows 10 or later, or macOS 10.14 or later, or most Linux distributions

## 🛠 Features Overview

Aar-Act offers several key features to help you secure infrastructure effectively:

- **Step-by-step security checklists:** Guided lists that cover essential protection measures  
- **Configuration templates:** Files you can use to set system parameters safely  
- **Incident response guides:** Clear instructions on how to react to ransomware or attacks  
- **Community tools:** Links and resources for cooperation within Senegal’s cybersecurity diaspora  
- **Updates and alerts:** Notifications when new threats or improvements arise  

These help you manage security without needing to understand technical details deeply.

After opening Aar-Act, start by selecting the area you want to secure. Common categories include:

If you get stuck, the built-in help menu explains terms plainly and offers next steps.

## 🎯 Best Practices for Security

Aar-Act promotes practical advice like:

- Regularly updating your software and operating system  
- Using strong, unique passwords and changing them often  
- Limiting access to important systems to authorized people only  
- Backing up critical data offline and online  
- Watching for suspicious emails or files and not opening unknown attachments  
- Training all users to recognize potential threats and report issues  

Following these steps can greatly reduce the risk of cyber attacks.
